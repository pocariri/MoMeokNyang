# 모먹냥 (MoMeokNyang)
"오늘 뭐 먹지?" 결정 장애를 해결하고 주변 맛집까지 한 번에 찾아주는 유저 맞춤형 iOS 애플리케이션

본 프로젝트는 사용자가 일상에서 겪는 메뉴 선택의 고민을 재미있는 랜덤 추천 방식으로 해결하고, 결정된 메뉴를 기반으로 주변의 실제 음식점을 지도상에
표시 및 저장할 수 있도록 돕는 실용적인 라이프스타일 앱입니다.


## 프로젝트 소개 & 주요 기능

### 사용자 인증 및 프로필 관리
- Firebase Authentication을 기반으로 한 안전하고 간결한 회원가입 및 로그인 시스템을 제공합니다.
- 사용자는 자신만의 고유한 닉네임을 설정하여 개인화된 경험을 제공받습니다.

### 음식 메뉴 랜덤 추천 (카테고리 기반)
- 데이터베이스(Firestore)에 등록된 다양한 음식 데이터 중 실시간으로 무작위 메뉴를 추출하여 유저에게 제안합니다.
- 마음에 들지 않을 경우 '한 번 더 랜덤(Retry)' 기능을 통해 즉시 새로운 메뉴를 추천받을 수 있습니다.
- 당첨된 메뉴에 어울리는 고화질 음식 사진(Asset Match)이 화면에 실시간으로 매칭되어 시각적 만족도를 높였습니다.

### Apple MapKit 기반 주변 맛집 실시간 검색
- 추천된 메뉴를 확정하는 순간, 유저의 현재 위치 정보를 기반으로 반경 1km 내의 실제 음식점을 탐색합니다.
- CoreLocation 및 MKLocalSearch 엔진을 활용하여 신뢰도 높은 애플 지도 데이터를 수신하고 지도 위에 커스텀 민트색 핀(Annotation)을 뿅뿅 표시합니다.

### 맛집 찜하기 & 히스토리 기록 관리
- 지도에 표시된 가게가 마음에 들면 하트 버튼을 눌러 나만의 찜 목록(`SavedRestaurants`)에 주소 및 추천 메뉴와 함께 영구 저장합니다.
- 내가 최종 결정했던 메뉴들은 시간순으로 최근 기록(`EatingRecords`) 탭에 차곡차곡 쌓여, 언제든 과거에 무엇을 먹었는지 직관적인 세그먼트 테이블뷰 리스트로 확인할 수 있습니다.

## 2. 기술 스택 & 개발 환경 (Tech Stack)

- 언어 (Language): Swift 5
- UI 프레임워크: UIKit (Storyboard 기반 오토레이아웃 설계)
- 백엔드 & 데이터베이스: Firebase Auth, Firebase Cloud Firestore
- 지도 & 위치 서비스: Apple MapKit Framework, CoreLocation Framework
- 버전 관리: Git, GitHub
- 개발 환경: Xcode 17.3 / iOS 17.0+ (시뮬레이터 타겟: iOS 26.3)


## 3. 빌드 및 실행 가이드

필독 사항:  본 저장소에는 시뮬레이터 빌드 크래시를 방지하기 위해 파이어베이스 연동 필수 파일인 `GoogleService-Info.plist`가
기본으로 포함되어있으므로, 클론 후 별도의 세팅 없이 즉시 빌드가 가능합니다.

앱을 정상적으로 구동하기 위한 순서는 다음과 같습니다.

### 시뮬레이터 가상 위치(GPX) 설정 (★필수)
본 앱은 위치 기반 맛집 검색 앱이므로, 시뮬레이터의 기본 위치에서는 한국 음식점 검색이 제한됩니다. 원활한 심사를 위해 아래 설정을
반드시 적용해 주세요.

1. Xcode에서 앱을 빌드 및 실행(`Command + R`)합니다.
2. 시뮬레이터가 켜지면 Xcode 하단 디버그 영역(콘솔 창 위쪽)에 있는 나침반 모양 아이콘(Location 변경 버튼)을 클릭합니다.
3. 목록에서 프로젝트에 내장된 `Seoul` GPX 항목을 선택해 줍니다. 

이 작업을 통해 시뮬레이터의 위치가 서울 좌표로 세팅되어 정상적인 한국 맛집 데이터를 애플 서버로부터 수신하게 됩니다.
